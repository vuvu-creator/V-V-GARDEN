# Website bán hoa có Decap CMS

Trang quản trị sau khi cài đặt: `https://TEN-WEBSITE.netlify.app/admin/`

Dữ liệu được Decap CMS quản lý:
- `content/site.json`: thông tin cửa hàng và nội dung trang
- `content/products.json`: danh sách sản phẩm
- `assets/uploads`: ảnh tải từ trang quản trị

Xem file `HUONG-DAN-DECAP.txt` để cài đặt từng bước.
